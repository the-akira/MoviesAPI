About

Introduction

Welcome to the Movies API!

Movies API seeks to bring resources related to the Movies Universe.

You can explore a wealth of data about movies, like directors, actors and genres.

The Idea

The idea of the Movies API project is to provide a service that provides concise data about the Movies Universe and a simple way to interact with the data and integrate it into your application.

The data is available in JSON format and is exposed through a REST implementation that allows you to collect the data programmatically using any programming language or some other specialized tool.

Features

We are using technologies like Django and Django REST framework to offer you quality and cool features like Pagination and Search Filter!
